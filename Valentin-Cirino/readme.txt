Must download the file FAOSTAT_data_1-10-2022.csv from the database zip folder and import it. The file had to be put in a zip folder since it was too big for GitHub.
The first two visualisations simply need to be ran. 

The temperaturedeviation_Valentin.py program requires 3 user inputs which are labelled in the console. 
1: A country
2: A temperature value (can be negative)
3: Whether you are looking for values above or below

It will then show the instances of the temperature changes being superior or inferior to this value in that specific country.
